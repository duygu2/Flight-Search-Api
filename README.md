# Flight Search API

Flight Search API, Developed backend API for a flight search application.


## Used Technologies 

- Java 17
- Spring Boot 3.2.2
- Spring Data JPA
- MySQL
- JSON Web Token (JWT)
- Springdoc OpenAPI (Swagger)
